Create Question

To create a new multiple-choice question go to the 'Create Question' page. Enter the question and four possible answers. Select the correct answer.


Create Quiz

To create a new quiz, go to the 'Create Quiz' page and select your questions. Save with an unique quiz name.


Play

To play a quiz, go to 'Play' and select the quiz name from the drop down box. To play, click on your chosen answer. The correct answer is shown with a 'tick'. And your score will update. Press 'Next' button to get the next question.


Results

To see recent quiz results, go to the 'Results' page and select the quiz name from the drop down box. Results of the last five quiz contestants are shown as a barchart.


Log Out

Click on 'Log Out', to exit the quiz.
